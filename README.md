# ПЕО для г. Шымкент

Границы первичных единиц опроса (ПЕО) для исследования пенсионной системы в г. Шымкент.

## 📍 Что это?

Каждый файл — это карта одного избирательного участка (ПЕО), где будет проводиться опрос.

## 📂 Структура репозитория

### GeoJSON файлы
- **Имя файла:** `номер_ПЕО.geojson`
- **Пример:** `12.geojson` — границы ПЕО №12

### Карточки для интервьюеров
- **Папка:** `карточки/`
- **Содержание:** Карточки сегментов для каждой ПЕО

Каждая карточка содержит:
- Границы сегмента (адреса)
- Стартовую точку для начала обхода
- Размер шага
- Субквоту (целевое количество интервью)

## 🗺️ Как посмотреть карту?

Откройте любой файл `.geojson` на GitHub — карта появится автоматически.

## 📝 Примечание

Файлы используются для:
- Планирования маршрутов интервьюеров
- Сегментации территории для random walk
- Контроля покрытия территории
